Directories

Here you can see all the relationships between the current level and the other levels in the system.
The concept of level is like a directory here. By default (only for demo), directories are geographically distributed and connected with each other as a binary tree-like structure. You can also think of different levels as different websites. Sometimes you can visit a website using your gmail account. Gmail is like one level here, and the website is a different level.
